Мини систем за банкомат
Опис:
Напиши функција која ќе побара:

Салдо на сметка

Износ за повлекување

Правила:

ако износот е поголем од салдото → „Недоволно средства“

ако износот е помал или еднаков на салдото → пресметај ново салдо

ако износот е негативен → грешка

Пример:



Салдо: 5000
Износ: 2000
Резултат: Ново салдо: 3000
